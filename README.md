# In-Memory Transaction Database

A simple key-value database with transaction support. Keys are strings, values are integers.

## How to Run

### Requirements

- Python 3.10 or newer

### Steps

1. Clone this repository or download the source files:

   ```bash
   git clone <YOUR_REPO_URL_HERE>
   cd in-memory-transaction-db
   ```

2. Run the program:

   ```bash
   python main.py
   ```
